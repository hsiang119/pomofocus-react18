# [番茄鐘 Pomodoro Technique | 個人作品 Side Project]

## 簡述 - introduce

番茄工作法是一種時間管理方法，在1980年代由Francesco Cirillo創立。該方法使用一個定時器來分割出一個一般為25分鐘的工作時間和5分鐘的休息時間，而那些時間段被稱為pomodoros，為義大利語單詞 pomodoro（中文：番茄）。
